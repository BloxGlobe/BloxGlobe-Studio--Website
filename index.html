How do I install Froststrap?
Download the latest release from the Releases page and run the installer included.
Is Froststrap safe to use?
Yes, Froststrap is a open-source project, You can check the repository for the code.
Where can I report issues or request features?
Open an issue on the GitHub repository. Whiling providing the log files and a way to reproduce the issue. If you want a faster way, then create a support/bug report forum on our discord
